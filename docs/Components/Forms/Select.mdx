Select component allow users pick a value from predefined list of options.

### Import
```js title=" Select.js" bash
import {Select} from  @kineticGlass/kinetic-glass-ui
```
### Usage
It is an example of a basic Select.
```js title="Select.js" bash
<Select
  options={[
    {
      label: 'Chocolate',
      value: 'chocolate'
    },
    {
      label: 'Strawberry',
      value: 'strawberry'
    },
    {
      label: 'Vanilla',
      value: 'vanilla'
    }
  ]}
  width="250px"
/>
```
